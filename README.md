# FDEProject

Campaign dashboard project with a Next.js frontend (`dashboard/`) and Campaign API documentation (`campaign_api_docs.md`).

## Repository Layout

- `dashboard/`: Next.js app for listing and inspecting campaigns
- `campaign_api_docs.md`: reference documentation for Campaign API endpoints and parameters

## Dashboard Documentation

See `dashboard/README.md` for:

- Local setup and run instructions
- API client usage and environment variables
- Current dashboard UX behavior (card list + modal detail)
- Timeout/retry resilience policy
- Client-side filter validation rules
- Metric and count derivation details
